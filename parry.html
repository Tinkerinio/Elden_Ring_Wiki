Parrying

Parrying is a gameplay mechanic in Elden Ring.


Parrying is a specific type of Ash of War/weapon Skill action. Usually found on shields, but not exclusively.


To successfully parry an attack means to deflect an incoming blow, which then stuns the attacker and leaves them open to a counterattack. There is also a small number of reusable Tools that can deflect specific attacks, although this is not strictly a "parry".


Most Enemies in the game have some number of attacks that can be deflected via parry, including many of the Bosses, but there is no visual queue or sound effect that makes it obvious whether or not an attack can be parried.
